Lab Description
PassGate is a hands-on password security lab that demonstrates critical authentication vulnerabilities. Learn how weak passwords and information leakage can lead to complete system compromise.

Difficulty: Beginner
Category: Password Security
Author: Ayed A. Areda
GitHub: AyedAA

Quick Start
Download all lab files to a folder

Open Command Prompt and navigate to the folder:

cmd
cd C:\path\to\PassGate-lab
Install dependencies:

cmd
npm install
Run the lab:

cmd
node server.js
Access http://localhost:3000 in your browser

Files Included
index.html - Login page

dashboard.html - Dashboard with flag

server.js - Vulnerable server

package.json - Dependencies

Objective
Exploit password security vulnerabilities to capture the hidden flag.
